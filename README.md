# Portfolio Pro

Professional Portfolio with Django + Tailwind + API + Realtime.

## Features

✅ Admin Panel (`/cms/`)  
✅ Project Management (Programming, 3D, Multimedia)  
✅ Uploads (images, videos, 3D `.glb`, `.gltf`, `.fbx`)  
✅ Export PDF / ZIP  
✅ API REST `/api/projects/`  
✅ Responsive UI with Tailwind  
✅ Dockerized for easy deploy

## How to run

```bash
git clone <your-repo-url>
cd portfolio_pro
cp .env.example .env
docker-compose up --build
```
